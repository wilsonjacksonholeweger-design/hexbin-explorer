# HexBin

HexBin é um projeto desenvolvido em C para estudar manipulação de arquivos binários.

## Funcionalidades

- Abre qualquer arquivo em modo binário.
- Lê o arquivo em blocos de 16 bytes.
- Exibe o offset de cada bloco.
- Converte os bytes para hexadecimal.
- Gera um dump em `saida.txt`.

## Tecnologias

- Linguagem C
- GCC

## Exemplo

00000000 | 4E 45 53 1A 08 00 21 02 00 00 00 00 00 00 00 00
